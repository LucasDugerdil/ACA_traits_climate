## Description
This script loads and cleans community-weighted mean (CWM) trait values devired from both vegetation plots and pollen surface samples from the Arid Central Asia (ACA).
Then, the relationships between CWMs and climate parameters are tested for both datasets and compared. This script also assess spatial representativeness of *WorldClim2.1* (Fick and Hijmans, 2017) and *CHELSA* (Karger et al., 2017).
The methodological study to reconstruct CWM from pollen data is available from Dugerdil et al. (2025a, JBI) and the associated script may be found in [https://github.com/LucasDugerdil/TraitPollen][https://github.com/LucasDugerdil/TraitPollen]

## Structure
- `/` main.R – the main R script to run
- `Import/Script/` – original script with the R packages to load and the functions developped for this study
- `Import/Data/` – original raw datasets (not modified)
- `Import/Data/Climate/` – *CHELSA* and *WorldClim2.1* extrations for the ACA
- `Import/Data/Climate_simulations` – *TraCE-21K* simulation extracted at the location of the 6 lakes from the Caucasus range
- `Import/Data/CWM/` – CWM traits data for the vegetation plots and the pollen surface samples (both fine and coarse aggregation schemes)
- `Import/Data/Pollen/` – Pollen data, both surface samples and past sequences from the Caucasus
- `Import/Data/Spatial/` – ACA borders in shapefiles 
- `Import/Data/Traits/` – raw trait data from *TRY* and the matching table to assign trait to pollen types
- `Import/Data/Vegetation/` – raw vegetation plots from sPlot for the ACA
- `Figures/Article` – all figures generated by the `main.R` and present in the final article
- `Figures/SI` – all figures generated by the `main.R` and present in the Supplementary Information of the final article 
- `Results/` – RDS files of the results of the statistical analyses

## Run the script
1. Install [R](https://larmarange.github.io/analyse-R/installation-de-R-et-RStudio.html)
2. It is easier to use [Rstudio](https://posit.co/downloads/)
3. Download this GitHub repository from ZIP file (by clicing on the green button `<> Code` beyond. 
4. Open the `ACA_traits_climate.Rproj` file in Rstudio

5. Package installation
	- All analyses were performed in **R** using multiple packages loaded from the `source("Import/Script/setup_packages.R")` line. 
	- Verify you have the needed packages installed
	- If not use the command `install.packages("reshape2")` for instance

6. Change the Boolean variables into the `Run the script` section (e.g., `Import = T`) to import, reconstruct past environmental variables and drow the figure related to the article  Dugerdil et al. (in prep. for JoE)
